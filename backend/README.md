## Description

- Service based on [Nest](https://github.com/nestjs/nest)

# Requirement
- Node 12

## Installation

```bash
$ npm install
```

## Lint

```bash
$ npm run lint
```

## Run the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
